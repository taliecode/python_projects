# Python CLI Calculator

A simple calculator that runs in the terminal. Enter two numbers, pick an operator, and it gives you the result. You can keep doing calculations until you're done.

---

## Features

- Addition, subtraction, multiplication, and division
- Loops so you can do multiple calculations in one session
- Handles divide by zero
- Gracefully rejects invalid operators

## Getting Started

Make sure you have Python 3 installed, then run:

```bash
python calculator.py
```

## Usage

```
Enter first number: 10
Enter second number: 5
Enter operator (+ - * /): *
10.0 * 5.0 = 50.0
Next calculation? (y/n): n
Goodbye!
```
